# Rotworm Killer

A simple browser-based game inspired by Tibia where players click rotworms to score points.

## How to play
1. Open `index.html` in a web browser.
2. Click on green worms to "kill" them and increase your score.
3. More worms spawn over time. Keep clicking!

## Tech stack
- HTML
- CSS
- JavaScript (canvas)

## Development
Just edit the files in `/js` and refresh your browser. No build steps required.
